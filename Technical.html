Chillennium Game Jam: Heist.exe

This project was a part of an called Chillennium, a game jam where participants are tasked with making a video game in 48 hours based on a prompt given at the beginning. As the lead programmer, I was tasked with:

  • creating the camera angle for the gameplay
  • implimenting the movement of the characters and enemies on the screen based on user input
  • developing interactions between the player character and the enemies/surroundings
  • etc.

Hangman Website

This is a project that looks like a fun yet challenging way to familiarize yourself with website making. It combines all 3 elements of basic website design:

  • HTML for the fundamental website design
  • CSS for the style/design of the website (the frontend development)
  • JavaScript for the inner functionality of the game (the backend development)
Though this is not a project I am currently working on, I think it would be a good way to get more comfortable with website design that I can impliment into future projects.
